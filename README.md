# 📊 Petrochemical Refinery LDAR VOC Emission Analysis Dashboard
A Power BI dashboard project for analyzing VOC emissions and Leak Detection & Repair (LDAR) performance in a petrochemical refinery environment.

## 📖 Project Overview
This dashboard helps monitor:  
•	VOC emission trends   
•	Leak severity distribution   
•	High-risk leaking components   
•	Equipment-wise emission contribution   
•	Chronic leaking tags   
•	Plant-wise emission performance   
The project focuses on environmental monitoring, maintenance prioritization, and operational analytics using Power BI.  

## 📂 Dataset
### Table 1 — Plant_Data  
Contains operational LDAR monitoring data:  
•	Date   
•	Component Tag No.   
•	Equipment   
•	Components   
•	Plant   
•	Screening Value (ppm)   
•	VOC Emission (Kg/day)   
### Table 2 — Dates  
Custom calendar table used for time intelligence analysis:  
•	Date   
•	Year   
•	Quarter   
•	Month Name   

## 🎯 Key KPIs
•	Total VOC Emission   
•	Leak Count   
•	High Severity Leaks   
•	VOC Reduction %   
•	Inspection Coverage   
•	Chronic Leaking Components   

## 📈 Dashboard Features
•	VOC emission trend analysis   
•	Equipment-wise VOC ranking   
•	Leak severity monitoring   
•	Top leaking component identification    
•	Plant-wise VOC heatmap   
•	Chronic leak tracking   
•	Interactive slicers and filters
